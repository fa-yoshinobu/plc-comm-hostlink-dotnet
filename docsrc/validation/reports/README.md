# Validation Reports

This folder contains hardware validation and compatibility test reports for `PlcComm.KvHostLink`.

Each report documents the test conditions, target hardware, and results for a specific verification session.

## Naming Convention

```
<TOPIC>_<DATE>.md
```

Example: `KV7500_SMOKE_TEST_2026-03-21.md`

## Status Labels Used in Reports

- `verified`: behavior confirmed on current hardware
- `not tested`: not yet verified on hardware
- `deferred`: known gap, not blocking current release
